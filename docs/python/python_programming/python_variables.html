# 파이썬 변수

파이썬 프로그래밍에서 변수는 데이터 값을 저장하기 위한 예약된 메모리 위치입니다. 다른 언어와 달리 파이썬에는 변수를 선언하는 키워드가 없습니다.

파이썬의 다양한 데이터 유형에는 숫자, 문자열, 리스트, 튜플, 집합, 사전 등이 있습니다. Python 변수는 특정 데이터 유형으로 선언할 필요가 없으며 초기화 후에도 유형을 변경할 수 있습니다.

다음 예제는 이를 보여줍니다:

\# 변수 테스트 1
x = 25 # 정수 값
print("x =", x)
x = "India" # 문자열 값
print("x =", x)
x = 52.7 # 부동 소수점 값
print("x =", x)

출력합니다:

x = 25
x = India
x = 52.7

# 파이썬 변수에 대한 규칙

파이썬 변수에 대한 규칙은 다음과 같습니다:

  • 변수 이름은 문자 또는 밑줄(_)로 시작해야 합니다.
  • 변수 이름은 숫자로 시작할 수 없습니다.
  • 변수 이름에는 영숫자와 밑줄 문자(A-z, 0-9 및 _)만 포함할 수 있습니다.
  • 파이썬에서 변수 이름은 대소문자를 구분합니다(년, 년, 년은 서로 다른 세 가지 변수입니다).

텍스트와 변수를 결합하려면 파이썬에서는 '+' 문자를 사용합니다. 숫자의 경우 '+' 문자는 수학 연산자로 작동합니다. 아래 예제를 참조하세요:

# 변수 테스트 2
x = "Great"
print("파이썬은 " + x)
x = 15
y = 10
print(x + y)

출력합니다:

파이썬은 Great
25

# 여러 변수를 함께

파이썬에서는 한 줄에 여러 변수에 값을 할당할 수 있습니다. 또한 한 줄에 여러 변수에 같은 값을 할당할 수도 있습니다. 아래 코드를 참조하세요:

# 변수 테스트 3
a, b, c, d = "Delhi", "Kolkata", "Mumbai", "Chennai"
print(a, b, c, d)
a = b = c = d = "India"
print(a, b, c, d)

출력:

Delhi Kolkata Mumbai Chennai
India India India India India
