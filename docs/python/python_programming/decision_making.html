# 의사 결정

파이썬 프로그래밍에서 조건문은 특정 조건이 참인지 거짓인지에 따라 다른 계산이나 동작을 수행합니다. 이를 바탕으로 특정 결정을 내려야 합니다.

기본적으로 조건문은 파이썬 _if_문으로 처리됩니다. 이는 때때로 elif 또는 else 문과 연관되기도 합니다.

이러한 문이 제대로 작동하려면 중괄호를 사용하는 대신 공백을 사용하여 코드에서 범위를 정의하는 들여쓰기를 사용해야 합니다. 다음 예제는 이를 보여줍니다:

# 조건부 테스트 1 a = 100 b = 66 if a > b: print("a가 b보다 큽니다") elif a == b: print("a와 b는 같습니다") else: print("a는 b보다 작습니다")

출력합니다:

a는 b보다 큽니다.

다음은 주어진 세 개의 정수 중 가장 큰 것을 찾는 중첩된 if 문을 보여주는 예제입니다:

# 조건 테스트 2 a = 80 b = 66 c = 95 IF A > B: if a > c: print("a가 가장 크다") else: print("c가 가장 크다") else: if b > c: print("b가 가장 크다") else: print("c가 가장 크다")

출력합니다:

c가 가장 큽니다.
