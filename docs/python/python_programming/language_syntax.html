# 언어 구문

파이썬 프로그래밍 언어는 사용하기 쉬운 간단한 구문을 가지고 있어 컴퓨터 프로그래밍을 처음 배우려는 사람에게 완벽한 언어입니다. 파이썬 구문 규칙은 다음과 같습니다.

1. 명령 종결자 없음: 파이썬에는 세미콜론이 없는 C, C++, Java 프로그래밍 언어와 달리 명령 종결자가 없습니다.

print("Hello World!")
  1. 코드 들여쓰기: 클래스 및 함수 정의 또는 흐름 제어를 위한 코드 블록을 표시하기 위해 중괄호 { }를 제공하지 않습니다(예: if...else, while, for 등). 코드 블록은 줄 들여쓰기로 표시되며, 이는 엄격하게 준수됩니다.
if a > b:
   print("a가 b보다 큽니다")
else:
   print("a는 b보다 작습니다")
  1. 파이썬 따옴표: 파이썬 프로그래밍에서는 작은따옴표 '', 큰따옴표 "", 심지어 삼중 따옴표 ''' """를 사용하여 문자열 리터럴(즉, 문자열 상수)을 나타낼 수 있습니다.
word = '단어'
sentence = "이것은 문장입니다."
paraph = """이것은 단락입니다.
여러 줄과 문장을 포함합니다."""
  1. 파이썬 주석: 파이썬에서는 코드의 가독성을 높이기 위해 프로그램 시작 부분에 기호 #을 사용하여 _주석_을 작성할 수 있습니다. 주석은 프로그램의 어느 곳에나 배치할 수 있습니다. 주석은 파이썬 코드가 실행되는 동안 무시됩니다.
# 이것은 주석입니다.

큰따옴표로 묶은 문자열을 사용하여 여러 줄로 주석을 달 수도 있습니다:

'''
이것은 여러 줄의
주석입니다.
'''

5. 여러 줄 문: 파이썬에서는 줄 이어 문자( \ )를 사용하여 여러 줄 문을 나타낼 수 있습니다.

합계 = 100 + \\
      200 + \\
      300

6. 한 줄에 여러 명령문: 한 줄에 두 개 이상의 실행문을 별도로 작성할 수 있으며, 세미콜론 ;을 사용하여 이러한 문을 구분해야 합니다.

print("첫 번째 줄"); print("두 번째 줄"); print("세 번째 줄");
