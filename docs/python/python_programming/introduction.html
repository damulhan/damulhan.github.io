# 파이썬 소개

파이썬은 해석된 프로그래밍 언어로, 프로그래머는 IDE(예: Spyder) 또는 텍스트 편집기(예: Notepad++)에서 파이썬(.py) 파일을 작성한 다음 해당 파일을 파이썬 인터프리터에 넣어 실행해야 합니다.

# 옵션 1:

이미 언급한 바와 같이, Python 코드는 Anaconda라는 오픈 소스 소프트웨어 플랫폼의 일부인 Spyder IDE를 사용하여 작성됩니다. Anaconda에는 Python, Spyder, IPython 및 기타 다양한 데이터 분석 라이브러리가 포함되어 있습니다. 따라서 www.anaconda.com에서 Anaconda를 다운로드한 후 MS Windows PC에 설치하는 것을 권장합니다. 여기서 PATH 환경 변수 값 설정은 선택 사항입니다.

스파이더 IDE의 에디터에서 test.py라는 작은 파이썬 코드(print() 함수로 일부 메시지를 출력하는 줄 코드 중 하나)를 작성했습니다. 여기에서 코드를 확인하세요:

print("Hi from techguruspeaks.com!")

# 옵션 2:

또는 www.python.org에서 Python 인스톨러를 다운로드하고 설치 디렉토리를 PATH 환경 변수에 추가할 수도 있습니다(필수). 이 경우 Windows PC에 파이썬이 설치되어 있는지 확인하려면 명령줄에서 다음을 실행합니다(여기서 '$'는 프롬프트를 의미함):

python --version

이제 명령줄을 열고 프로그램을 저장한 디렉토리로 이동한 다음 명령줄에서 실행합니다:

python test.py
